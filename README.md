# PowerShell IT Automation Scripts

## Project Overview
A collection of PowerShell automation scripts to perform common IT support tasks including system health monitoring, disk space checking, automated backups, user account reporting and network diagnostics.

## Technologies Used
- Windows 10
- PowerShell ISE
- Windows Management Instrumentation (WMI)
- PowerShell Cmdlets

## Scripts

### 1. System Health Report
Checks CPU, RAM, disk space and uptime and exports a full report to a text file.

### 2. Disk Space Monitor
Monitors all drives and generates a warning when free space falls below 20%.

### 3. Automated Backup Script
Backs up a source folder to a timestamped destination folder and logs the result.

### 4. User Account Report
Generates a report of all local user accounts including their status and last logon time.

### 5. Network Diagnostics Report
Runs a full network diagnostic including IP configuration, DNS servers and connectivity checks.
